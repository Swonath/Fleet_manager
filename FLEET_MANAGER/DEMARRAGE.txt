?? COMMENCER IMMÉDIATEMENT
???????????????????????????????????????????????????????????????????????????

ÉTAPE 1: Installer la base de données
?????????????????????????????????????
1. Ouvrir MySQL Command Line ou MySQL Workbench
2. Exécuter:
   mysql -u root -p < FLEET_MANAGER/Database/fleet_manager.sql

ÉTAPE 2: Configurer l'application
???????????????????????????????????
1. Ouvrir FLEET_MANAGER/Config/DatabaseConfig.cs
2. Adapter les paramètres si nécessaire:
   - Server: localhost
   - Database: fleet_manager
   - UserId: root
   - Password: (votre mot de passe MySQL)
   - Port: 3306

ÉTAPE 3: Lancer l'application
?????????????????????????????
cd FLEET_MANAGER
dotnet build
dotnet run

ÉTAPE 4: Se connecter
????????????????????
- Username: admin
- Password: admin123
