?? OPTION 1: MySQL Workbench (RECOMMANDÉ)
?????????????????????????????????????????????????????????????????????????

ÉTAPE 1: Ouvrir MySQL Workbench
???????????????????????????????
1. Lancer MySQL Workbench
2. Connexion: Double-cliquez sur votre connexion MySQL local

ÉTAPE 2: Ouvrir le fichier SQL
???????????????????????????????
1. Menu: File ? Open SQL Script
2. Naviguer vers: FLEET_MANAGER\Database\fleet_manager.sql
3. Cliquer "Open"

ÉTAPE 3: Exécuter le script
???????????????????????????????
1. Le script s'affiche dans l'éditeur
2. Menu: Query ? Execute All
   OU Raccourci: Ctrl+Shift+Enter
   OU Bouton: ? Exécuter (en haut à droite)

ÉTAPE 4: Vérifier l'exécution
???????????????????????????????
1. Regarder la sortie (Output panel en bas)
2. Doit afficher: "X queries executed successfully"
3. Aucune erreur en rouge

ÉTAPE 5: Vérifier les données
???????????????????????????????
1. Panel gauche: Cliquer sur "Refresh" (F5)
2. Développer: fleet_manager ? Tables
3. Clic droit sur "vehicules" ? Select Rows - Limit 1000
4. Vous devez voir 5 véhicules

?????????????????????????????????????????????????????????????????????????

ÉTAPE 1: Ouvrir Command Prompt/PowerShell
??????????????????????????????????????????
- Windows: Cmd ou PowerShell
- Mac/Linux: Terminal

ÉTAPE 2: Se connecter à MySQL
??????????????????????????????
Pour MySQL 8:
```bash
mysql -u root -p
```

Puis entrer le mot de passe quand demandé

ÉTAPE 3: Exécuter le script
???????????????????????????
Une fois connecté, taper:
```bash
source "C:\Users\[VotreNom]\Desktop\Projet_lourd\FLEET_MANAGER\Database\fleet_manager.sql";
```

Ou si vous êtes dans le dossier:
```bash
source FLEET_MANAGER/Database/fleet_manager.sql;
```

? SI ERREUR: "Database already exists"
?????????????????????????????????????????????????????????????????????????

1. Supprimer l'ancienne base de données:
   ```sql
   DROP DATABASE IF EXISTS fleet_manager;
   ```

2. Puis réexécuter le script complet

OU directement dans le script, il y a:
   ```sql
   DROP DATABASE IF EXISTS fleet_manager;
   CREATE DATABASE IF NOT EXISTS fleet_manager;
   ```
   Donc pas de soucis!
